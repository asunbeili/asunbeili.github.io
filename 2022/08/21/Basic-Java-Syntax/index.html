Basic Java Syntax

Java基础语法

注释

  • 单行注释 //
  • 多行注释 /* */
  • 文档注释 /** */

标识符

  • 必须以大小写字母_和$ 开头
  • 首字母之后可以用任意字母 数字 _和$字符组合
  • 不能使用关键字作为变量名和方法名
  • 标识符是大小写敏感的

数据类型

强类型语言

  • 所有变量严格按照要求
  • 所有变量必须先定义后才能使用

java数据类型分为两大类

  • 基本类型
    • 数值类型
      • 整数类型
        • Byte 占1个字节 -127-127
        • Short 占2个字节 -32768-32767
        • Int 占4个字节 正负21亿
        • long 占8个字节 无敌长
      • 浮点类型
        • float 占4个字节
        • double 占8个字节
      • 字符类型
        • char 占2个字节
    • boolean类型
      • 占1位，只有true和false两个值
  • 引用类型
